```
│   └── index.ts        # 导出文件

├── audio/              # 音频组件
│   ├── AudioPlayer.tsx      # 音频播放器
│   ├── AudioInfo.tsx        # 音频信息显示
│   ├── AudioConverter.tsx   # 音频转换器
│   ├── AudioDisplay.tsx     # 音频显示容器
│   ├── AudioWaveform.tsx    # 波形图组件
│   ├── AudioSpectrum.tsx    # 频谱图组件
│   └── index.ts             # 导出文件
├── video/              # 视频组件（现有）
└── media/              # 通用媒体组件

```

## 📝 使用指南

### 访问音频分析器
1. 打开主页
2. 点击 "Try Audio Analyzer →" 链接
3. 或直接访问 `/audio` 路由

### 分析音频文件
1. 点击 "Select Local File" 按钮
2. 选择音频文件（MP3, AAC, WAV等）
3. 查看元数据和播放音频

### 分析音频URL
1. 在输入框中输入音频URL
2. 点击 "Analyze" 按钮
3. 查看分析结果

### 转换音频格式
1. 上传音频文件
2. 切换到 "Converter" 标签
3. 选择输出格式和参数
4. 点击 "Convert"
5. 下载转换后的文件
